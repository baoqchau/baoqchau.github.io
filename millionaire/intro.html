Game Rules

Welcome to the game. Before we begin, what you need to know about the game is:

  • Each game has 10 questions. If you answer correct all 10, you get one million dollar.
  • The difficulty of the questions will increase after each one.
  • You have 30 seconds for each question. If you cannot answer after that time, you lose.
  • You can skip one question only ONCE. After that, you cannot skip any question.

If you are ready, now click the button below to start the game. Good luck!
